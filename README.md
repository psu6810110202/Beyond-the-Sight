# Beyond the Sight

เกม RPG Horror Puzzle สร้างด้วย Kivy Python

## เรื่องราว

**Beyond the Sight** เป็นเกมผจญภัยแนวสยองขวัญที่เล่าเรื่องราวของเด็กสาวที่ต้องเผชิญกับวิญญาณและความลี้ลับในโลกหลังความตาย ผ่านการเดินทางในแต่ละวันเพื่อช่วยเหลือวิญญาณที่ต้องการความช่วยเหลือ

## ผู้พัฒนา

- **เรื่องราว**: Parichat Hongsuwanwattana
- **ศิลปะ**:
  - Player: Parichat Hongsuwanwattana
  - NPC: Phunchita Thongkum
  - Reaper: Phunchita Thongkum
- **โค้ด**: Parichat Hongsuwanwattana และ Phunchita Thongkum

## วิธีการเล่น

### การควบคุม
- **WASD** - เดิน
- **Shift** - วิ่ง (ใช้ Stamina)
- **E** - โต้ตอบ/คุย
- **Enter** - ข้อความถัดไป
- **Q** - ใช้ไอเทม Blue Stone (สลายพลังศัตรู)
- **ESC** - เมนูหยุดเกม

### ระบบหลัก

#### ระบบการเคลื่อนที่
- เดินและวิ่งในแผนที่ 2D แบบ Grid-based
- ระบบ Stamina สำหรับการวิ่ง
- อนิเมชันการเคลื่อนไหวที่สมจริง

#### ระบบศัตรู
- **ศัตรูคงที่**: 4 ตัวในตำแหน่งที่กำหนด
- **ศัตรูสุ่ม**: 60 ตัวที่กระจายในโซนอันตราย
- ศัตรูจะไล่ตามผู้เล่นเมื่อเข้าสู่รัศมีตรวจจับ
- ระบบ Safe Zone รอบๆ Reaper

#### ระบบชีวิต
- มีหัวใจ 3 ดวง
- สูญเสีย 1 ดวงเมื่อชนศัตรู
- ตายทั้งหมดจะวาร์ปกลับไปยังจุดเริ่มต้น

#### ระบบเรื่องราว
- ผจญภัยข้าม 3 วัน
- ระบบบทสนทนากับ NPC และ Reaper
- ระบบเควสและการตัดสินใจ
- ปลายทางหลายแบบขึ้นอยู่กับการเลือก

#### ระบบไอเทม
- ค้นหาดาวใน Day 1
- รวบรวมชิ้นส่วนตุ๊กตาเพื่อเควสหลัก
- ไอเทมพิเศษเช่น Blue Stone สำหรับสลายพลังศัตรู

## จุดเด่นของเกม

### การออกแบบแผนที่
- แผนที่หลักขนาดใหญ่พร้อมโซนอันตราย
- ระบบค้นหาของในบ้าน
- การจำกัดการมองเห็นด้วยระบบหมอก

### กราฟิกและเสียง
- ศิลปะ Pixel Art สไตล์ Retro
- อนิเมชันตัวละครที่ลื่นไหล
- เสียงบรรยากาศที่เหมาะกับอารมณ์
- เสียงผีไล่ตามประเภทตัวละคร

### ปริศนาและการตัดสินใจ
- ปริศนาที่ต้องใช้ความคิด
- ระบบ Choice ที่ส่งผลต่อเนื้อเรื่อง
- หลายจบขึ้นอยู่กับการกระทำ

## วิธีการติดตั้งและรัน

### ข้อกำหนดเบื้องต้น
```bash
pip install kivy
pip install kivy-deps.angle
pip install kivy-deps.glew
pip install kivy-deps.gstreamer
pip install kivy-deps.sdl2
```

### การรันเกม
```bash
python main.py
```
